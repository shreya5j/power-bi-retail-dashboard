# Power BI Retail Dashboard

## Overview

This is my Power BI project where I analyzed retail business data to understand sales, profit, customers, and product performance.
The goal was to create an interactive dashboard that gives clear insights for decision-making.



## What I Built

The dashboard has 4 pages:

**1. Executive Dashboard**

* Shows total sales, profit, and growth
* Sales trend over time
* Region-wise performance


**2. Product Analysis**

* Sales by category and sub-category
* Top 10 products based on sales
* Product-level insights

**3. Customer Insights**

* Sales by customer segment
* Top customers
* Customer-level analysis

**4. Profitability Analysis**

* Sales vs Profit comparison
* Profit by region and category
* Profit trends



## Skills Used

* Data cleaning using Power Query

* Creating interactive dashboards
